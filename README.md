# Critical Data Literacy

**[View the Live Interactive Course Here](https://adeline-hub.github.io/feminist-data-literacy-mooc/)**

## Overview

This repository contains a 7-module Massive Open Online Course (MOOC) dedicated to **Critical Data Literacy**. 

Unlike traditional data courses that focus strictly on software mechanics, this curriculum interrogates data architecture. It examines how historical normalizations, database schemas, SQL query logic, and predictive algorithms (AI/IoT) structurally dictate which human realities are recorded and which are mathematically erased.

## Course Structure

The curriculum traces the complete data lifecycle:

* **Module 1:** Definitions \& Data Types
* **Module 2:** The Benefits of Data (and Structural Limitations)
* **Module 3:** What is a Database?
* **Module 4:** Interacting with Data (SQL \& Literal Execution)
* **Module 5:** What is Data Analysis? (Variable Selection)
* **Module 6:** Predictions in Data (AI, IoT, and Feedback Loops)
* **Module 7:** Trends in Data (Privacy \& Synthetic Generation)
* **Capstone:** Exploratory Data Analysis (EDA) on Gender Representation

## Technical Stack

This course and its interactive dashboards are built using:

* **[Quarto](https://quarto.org/):** For scientific and technical publishing (HTML rendering).
* **Python (`pandas`, `plotly`):** For data processing and interactive data visualization.
* **Custom CSS:** Minimalist UI prioritizing readability and analytical focus.
