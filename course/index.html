# Rethinking Data Objectivity

Welcome to the Data Literacy MOOC. 

In the modern digital economy, data is frequently presented as a pure, objective truth. However, data is not a naturally occurring phenomenon; it is mathematically structured, architected, and managed by human systems. 

This course is designed for professionals who wish to move beyond basic data collection. True data literacy requires understanding **how** to use data tools (databases, SQL, AI), while simultaneously recognizing **who** those tools were historically optimized for. 

By identifying rigid normalizations, male-oriented baselines, and omitted variables, we can transform structurally flawed datasets into genuinely objective, actionable insights.
